## The Problem

Many elderly individuals struggle to describe symptoms clearly during digital consultations.

Common challenges:

- Difficulty typing detailed complaints  
- Forgetting important symptom details  
- Using non-medical language  
- Feeling overwhelmed during telehealth calls  

This leads to miscommunication, delayed diagnosis, and stress.
